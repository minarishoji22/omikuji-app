# おみくじアプリ

　ボタンを押すと7種類のおみくじから今日の運勢を占えるアプリです。

## 公開URL

　[]おみくじアプリを開く]
## 使用技術

- HTML
- CSS
- JavaScript

## 実装した機能
- ボタンクリックイベント
- ランダム画像表示
- setIntervalによる抽選演出
- setTimeoutによる抽選停止

## 工夫した点
- HTML / CSS / JavaScriptをファイルごとに分けました。
- 画像がランダムに切り替わることで、おみくじを引いている感覚が出るようにしました。

## 今後追加したい機能
- ボタン連打防止
- 結果ごとのコメント表示
- スマホ向けデザイン改善
