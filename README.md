# おみくじアプリ

## 概要
HTML / CSS / JavaScriptで作成した簡易おみくじアプリです。

ボタンをクリックすると、おみくじ画像がランダムに切り替わり、3秒後に結果が確定します。

## 使用技術
- HTML
- CSS
- JavaScript

## 実装した機能
- ボタンクリックイベント
- ランダム画像表示
- setIntervalによる抽選演出
- setTimeoutによる抽選停止

## 工夫した点
- HTML / CSS / JavaScriptをファイルごとに分けました。
- 画像がランダムに切り替わることで、おみくじを引いている感覚が出るようにしました。
