# おみくじアプリ

　ボタンを押すと7種類のおみくじから今日の運勢を占えるアプリです。

## 公開URL

　[おみくじアプリを開く](https://minarishoji22.github.io/omikuji-app/)

## 使用技術

- HTML
- CSS
- JavaScript

## 使用方法

⒈ 「おみくじを引く」ボタンを押します。
⒉ 抽選中はおみくじがランダムに切り替わります。
⒊ 約3秒後に今日の運勢が表示されます。

## 実装した機能

- 7種類のおみくじの中から結果をランダムに選択
- setIntervalによる抽選演出
- setTimeoutによる抽選時間の制御

## 工夫した点

- HTML / CSS / JavaScriptをファイルごとに分けました。
- 画像がランダムに切り替わることで、おみくじを引いている感覚が出るようにしました。
